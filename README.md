# 🎟️ Eventbrite IR Project

An AI-powered Information Retrieval and Recommendation System for Eventbrite events using Machine Learning and Natural Language Processing techniques.

---

## 📌 Project Overview

This project aims to build an intelligent event retrieval and recommendation system by collecting and analyzing event data from Eventbrite. The system applies Information Retrieval (IR), Machine Learning (ML), and Natural Language Processing (NLP) techniques to improve event searching, recommendation, and analysis.

The project includes data scraping, preprocessing, feature extraction, sentiment analysis, summarization, and recommendation functionalities.

---

## 🚀 Features

* 🔍 Event Search Engine
* 🤖 AI-based Recommendation System
* 🧠 Sentiment Analysis
* ✨ Text Summarization
* 📊 Exploratory Data Analysis (EDA)
* 🧹 Data Cleaning & Preprocessing
* 📑 TF-IDF Feature Extraction
* 🌐 Web Scraping from Eventbrite

## 📈 Project Workflow

1. Collect Event Data using Web Scraping
2. Clean and preprocess text data
3. Extract textual features using TF-IDF
4. Build search and recommendation models
5. Apply AI features such as sentiment analysis and summarization
6. Visualize insights and evaluate results
